\# Apex Manufacturing, Inc.



\## Information Technology Department



\### User Access Request Form



\---



\## Purpose



This form is used to request new user access, modify existing access, or remove access to company systems, applications, shared resources, or network services.



All access requests must be approved by the employee's department manager before implementation by the Information Technology department.
