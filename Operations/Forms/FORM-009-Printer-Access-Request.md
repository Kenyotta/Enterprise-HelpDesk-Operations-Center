\# Apex Manufacturing, Inc.



\## Information Technology Department



\### Printer Access Request



\---



\# Purpose



This form is used to request access to existing network printers, install new printers, modify printer permissions, or remove printer access for employees at Apex Manufacturing, Inc.



Printer access is managed through Active Directory security groups and print server permissions to ensure secure and efficient access to company printing resources.
