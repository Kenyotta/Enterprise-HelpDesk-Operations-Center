\# Apex Manufacturing, Inc.



\## Employee Directory



\### Purpose



This document serves as the official employee directory for Apex Manufacturing, Inc.



It contains the users, departments, reporting structure, usernames, email addresses, office locations, and assigned computer assets used throughout the Enterprise Help Desk Operations Center.



All future documentation—including Active Directory administration, Help Desk ticket scenarios, Standard Operating Procedures (SOPs), cybersecurity investigations, Microsoft 365 administration, Azure projects, and automation scripts—should reference this directory to maintain consistency across the enterprise environment.



\---



\# Username Standard



Employee usernames follow the format:



```text

First Initial + Last Name



Example:



Emily White

ewhite



David Miller

dmiller

smitchell

```
